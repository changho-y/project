# BillMate (자취생 공과금 관리 서비스)

> **복잡한 공과금 관리를 한눈에, 실시간 공과금 조회 및 거주자 전용 커뮤니티 플랫폼**

---

## 📌 프로젝트 개요
* **캡스톤디자인**: 대학교 동기 3명과 함께한 프로젝트입니다. 벡엔드를 담당했고, API연결 및 서버, DB 관리를 동기 1명과 함께 구현했습니다.
* **개발 배경**: 대한민국 자취생들이 수도, 전기, 가스비 등 다양한 공과금을 앱 하나로 실시간 확인하고 관리할 수 있도록 기획되었습니다.
* **주요 목표**: 공과금 변화의 시각화와 개인정보 노출 없는 안전한 건물 내 커뮤니티 제공

## Tech Stack
| 구분 | 기술 스택 |
| :--- | :--- |
| **Frontend** | React Native (Expo) |
| **Charts** | React Native Chart Kit |
| **External API** | Daum Postcode (WebView) |
| **Backend** | Node.js |
| **Database** | Firebase Firestore |
| **Auth** | Firebase Authentication |

## 핵심 기능

### 1. 실시간 공과금 조회 및 비교
* **통합 관리**: 수도세, 전기세, 가스비를 별도의 고지서 없이 실시간으로 통합 조회할 수 있습니다.
* **데이터 시각화**: 월별 공과금 변화를 그래프로 제공하여 사용자가 소비 패턴의 차이를 직관적으로 느낄 수 있게 합니다.

### 2. AI 챗봇
* **AI 관리비 비서**: 공과금 관련 질문에 빠르게 답변해주고, 사용자의 입력 기록을 바탕으로 맞춤형 절약 팁, 이전 지출 분석, 패턴 기반의 소비 경고 등을 제공합니다.

### 3. 전입·전출 시 정산 계산 지원
* **목적**: 사용자 간 분쟁이나 비용 부담 문제를 줄일 수 있습니다.
* **계산**: 전입 또는 전출 날짜에 따라 공과금을 정확하게 나누어 계산하는 기능을 제공합니다.

### 4. 안전한 거주자 전용 커뮤니티
* **익명성 보장**: 세대주가 여럿인 빌라나 원룸에서 개인정보 노출 위험 없이 소통할 수 있는 창구를 제공합니다.
* **거주 인증**: 건물 내 실제 거주중인 사람들만 사용 가능한 폐쇄형 커뮤니티 기능을 지원합니다.
* **사용자간 소통 플랫폼**: 거주 인증으로 실제 건물 내 실제 거주중인 사람들만 사용 가능한 폐쇄형 커뮤니티 기능과 별개로
앱 내 커뮤니티를 통해 사용자들은 관리비 절약 팁, 공과금 상식, 주거 생활 정보 등을 공유할 수 있습니다.

### 5. 분쟁 증거 보관함
* **증거 보관함**: 거주 중 불미스러운 일 발생시 법적 효력을 위해 사진과 영상을 모을 수 있는 분쟁 증거 보관함을 제공합니다.
